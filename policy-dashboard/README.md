# 政策透明化ダッシュボード

日本国政府の政策運営の透明性を向上させ、納税者が政策情報へアクセスしやすくするためのウェブアプリケーションです。様々な政策の予算配分、執行状況、成果を視覚的に分かりやすく表示します。

## 概要

このダッシュボードは、行政事業レビューのデータを基に、政府の事業に関する以下の重要な側面を視覚的に表示します：

- 府省庁別の事業数と予算配分
- 事業の実績と目標達成状況
- 予算執行方法と支出先の透明性
- 行政事業レビューの評価結果と改善状況

全てのデータは「見える化」され、納税者が自分の税金がどのように使われているかを簡単に把握できるようになっています。

## 使い方ガイド

### 政策概要ダッシュボード

**納税者の関心：** 「政府全体でどのような事業が行われているのか？どの府省が多くの事業を持っているのか？」

1. **KPI概要カード**で政府全体の事業数を確認
   - 総事業数、新規事業数、終了事業数を一目で把握
   - 評価改善要事業数から、見直しが必要とされている事業の全体像を理解

2. **事業区分分布**グラフで事業の内訳を確認
   - 前年度からの継続事業、新規開始事業、終了事業の割合を把握
   - どのくらいの事業が継続的に行われているかを確認

3. **府省庁別事業数**のツリーマップで予算事業の府省庁別分布を視覚的に把握
   - どの府省庁が多くの事業を持っているかを面積の大きさで比較
   - 府省庁ごとの事業数と全体に占める割合を確認

4. **主要経費分布**の棒グラフで予算用途を確認
   - 予算がどのような経費区分で使われているかを把握
   - 上位10件の主要経費を比較し、政府の優先分野を理解

### 事業パフォーマンスダッシュボード

**納税者の関心：** 「事業は目標を達成しているのか？どの事業が成果を出しているのか？」

1. **指標種別の統計カード**で成果指標の種類と数を把握
   - アクティビティ（活動）、アウトプット（結果）、アウトカム（成果）の3種類の指標数を比較
   - 政府がどのような指標で事業を評価しているかを理解

2. **目標達成度分布**の散布図で事業の達成状況を把握
   - 横軸の事業IDと縦軸の達成率の関係から、全体的な達成傾向を理解
   - 達成率の分布から、目標設定の適切さや実行力を考察

3. **高パフォーマンス事業トップ5**と**低パフォーマンス事業ワースト5**のテーブルで具体的な事例を確認
   - 目標達成率の高い優良事業の詳細を把握
   - 目標未達の事業を確認し、改善の必要性を評価

### 予算執行ダッシュボード

**納税者の関心：** 「予算はどのように使われているのか？誰がお金を受け取っているのか？」

1. **契約方式別の統計カード**と**契約方式別件数**グラフで予算執行の透明性を確認
   - 一般競争契約、随意契約などの方式別の割合を把握
   - 競争性の高い契約方式がどれだけ採用されているかを評価

2. **支出先種別分布**のツリーマップで予算の行き先を把握
   - 株式会社、特殊法人、一般社団法人など、支出先の種類別の金額割合を確認
   - どのような組織に多くの予算が流れているかを視覚的に理解

3. **支出先上位20社**のテーブルで具体的な支出先を確認
   - 最も多くの予算を受け取っている組織を把握
   - 支出額の規模感と支出先の種類を確認

4. **支出先種別比率**の棒グラフで支出先の構成比を確認
   - 各種類の支出先が全体に占める割合を比較
   - 公的機関と民間企業の比率など、予算の流れの傾向を理解

### 政策改善トラッカーダッシュボード

**納税者の関心：** 「事業評価の結果、どのような改善が行われているのか？」

1. **レビュー結果の統計カード**と**レビュー結果分布**グラフで事業評価の全体像を把握
   - 「現状通り」「一部改善」「抜本的改善」「終了予定」「廃止」の各評価の件数と割合を確認
   - 行政事業レビューによってどれだけの事業が見直されているかを理解

2. **府省庁別改善率**の水平棒グラフで府省庁の改善傾向を比較
   - どの府省庁が積極的に事業改善に取り組んでいるかを把握
   - 改善率の高い府省庁と低い府省庁の差を確認

3. **改善事例ハイライト**のテーブルで具体的な改善内容を確認
   - 実際にどのような改善が行われているかの事例を把握
   - 改善内容の質と深さを評価

## 使い方ガイドページ

アプリケーション内には詳細な使い方ガイドページが組み込まれています。このページには、以下の内容が含まれています：

- **納税者の視点に立ったユースケース**：一般納税者、政策関心層、予算監視層、政策評価関心層など、異なるニーズを持つユーザーがどのようにダッシュボードを活用できるかを示しています
- **各ダッシュボードの機能概要**：各ダッシュボードの主要機能と特徴の簡潔な説明
- **ステップバイステップガイド**：分析のための具体的な手順と活用方法
- **データソースの説明**：使用しているデータの出所と前処理方法

使い方ガイドページには以下の方法でアクセスできます：
- 左側メニューの「使い方ガイド」をクリック
- 画面右上のヘルプアイコン（？マーク）をクリック

## より深い分析のためのフィルター機能

画面左側のサイドバーにあるフィルター機能を活用することで、特定の関心領域に絞った分析が可能です：

- **府省庁フィルター**: 特定の府省庁のデータのみを表示
- **事業区分フィルター**: 新規/継続/終了事業など特定の区分に絞り込み
- **主要経費フィルター**: 特定の経費区分のデータのみを表示
- **予算規模フィルター**: 特定の予算規模の範囲に絞り込み
- **レビュー評価フィルター**: 特定の評価結果のデータのみを表示

これらのフィルターを組み合わせることで、例えば「文部科学省の新規事業のうち、抜本的改善が必要とされている事業」などの具体的な分析が可能になります。

## データソース

このダッシュボードは、[行政事業レビュー見える化サイト](https://rssystem.go.jp/top)で公開されている行政事業レビューのデータを使用しています。

### 行政事業レビューとは

行政事業レビューとは、各府省庁が、予算や基金を用いて行う原則すべての事業を、毎年度、自ら点検し、その点検結果を公表する取組です。

EBPM(Evidence-Based Policy Making: エビデンスに基づく政策立案)の手法等を用いて、事業の目的・課題を踏まえて成果指標を設定し、外部の視点も活用して点検を行います。点検結果は翌年度の予算要求や事業の執行等に反映し、事業の効果的・効率的な実施につなげます。

**EBPM（エビデンスに基づく政策立案）とは**：政策目的、その目的達成のために効果的な政策手段、政策手段と目的の論理的なつながり（ロジック）を明確にし、さらに、このつながりの裏付けとなるデータ等のエビデンス（根拠）を可能な限り求めることで、政策立案・実施・評価を実証的なものとし、政策の実効性や効率性を高めていく手法。

### 使用データファイル

- 基本情報・事業概要等（1-2_RS_2024_基本情報_事業概要等.csv）
- 効果発現経路・目標・実績（3-1_RS_2024_効果発現経路_目標・実績.csv）
- 効果発現経路・目標のつながり（3-2_RS_2024_効果発現経路_目標のつながり.csv）
- 点検・評価（4-1_RS_2024_点検・評価.csv）
- 支出先・支出情報（5-1_RS_2024_支出先_支出情報.csv）

- デプロイ: GitHub Pages

## 開発・貢献

このプロジェクトは政府の透明性向上を目的としています。改善提案やバグ報告は歓迎します。

### 開発環境のセットアップ

```bash
# リポジトリのクローン
git clone https://github.com/yourusername/policy-dashboard.git
cd policy-dashboard

# 依存関係のインストール
npm install

# 開発サーバーの起動
npm start

# データの前処理（実際のCSVデータから集計JSONを生成）
node lambda/processData.js

# または、サンプルデータの生成
node lambda/processSample.js

# ビルドと公開
npm run deploy
```
